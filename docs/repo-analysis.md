# Краткая заметка по реализации

Проект был переведён из раннего NL2SQL-прототипа в самостоятельное приложение `Толмач`.

Ключевые изменения:

- добавлена авторизация, роли и JWT;
- добавлены чаты, сообщения, шаблоны, отчёты и админские логи;
- аналитический слой упрощён до демо-схемы `cities`, `orders`, `cancellations`;
- Text-to-SQL подключён к локальной Ollama `qwen3:4b` через отдельный provider-слой;
- UI перестроен в светлый чат-интерфейс с левой панелью;
- Docker Compose теперь поднимает только нужные сервисы: backend, frontend, PostgreSQL.
